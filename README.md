## SkillHub Pro - Online Learning Platform (Task-1) ##

## Project Overview

SkillHub Pro is a responsive online learning platform developed using HTML, CSS, and JavaScript. The website provides users with an engaging interface to explore courses, learn new skills, and enroll in programs. The project focuses on responsive design, interactive user experience, and modern web development practices.

## Features

* Responsive design for all devices
* Modern and clean user interface
* Course showcase section
* Interactive image/content slider
* Dark mode functionality
* Course enrollment form
* Mobile-friendly navigation
* JavaScript-based user interactions

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Student Management API (Task-2) ##

## Project Overview

Student Management API is a RESTful backend application developed using Node.js and Express.js. The API allows users to perform CRUD (Create, Read, Update, Delete) operations on student records and demonstrates backend development concepts such as routing, middleware, and API design.

## Features

* Add new students
* Retrieve all student records
* Retrieve student by ID
* Update student information
* Delete student records
* Middleware validation
* RESTful API architecture
* JSON-based data storage

## Technologies Used

* Node.js
* Express.js
* JavaScript
* JSON

## Project Structure

Student-Management-API/
│
├── controllers/
├── routes/
├── middleware/
├── data/
│   └── students.json
│
├── server.js
├── package.json
└── README.md

##  Installation

bash
git clone <repository-url>
cd student-management-api
npm install
npm start

The server will start on the configured port.
