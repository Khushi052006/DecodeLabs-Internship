# SkillHub Pro

A modern responsive learning platform website built using:

- HTML5
- CSS3
- JavaScript

## Features

- Responsive Design
- Mobile Navigation Menu
- Dark Mode
- Animated Counters
- FAQ Accordion
- Course Search
- Testimonials Slider
- Contact Form Validation
- Glassmorphism UI
- Modern Hero Section
