# Student Management API

A simple REST API built using Node.js and Express.js.

## Features

- Get All Students
- Get Student By ID
- Add Student
- Update Student
- Delete Student
- Input Validation
- Proper HTTP Status Codes

## Installation

npm install

## Run

node server.js

Server URL:

http://localhost:5000

## API Endpoints

GET /api/students

GET /api/students/:id

POST /api/students

PUT /api/students/:id

DELETE /api/students/:id
